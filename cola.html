Programação Web é o processo de criar e manter sites e aplicacões para a internet. Envollve escrever códigos em linguaguens de programação como HTML, CSS, JAVASCRIPT, PHP, entre outros.

Para comecar programar precisamos utilizar tag, mas o que é uma tag? na programação as tags são trechos de código que marca ou identifica algo, como incio e o sim de um elemento.

bora conhecer algumas tags em HTML e suas funcções

HTML - (Hyper Text Markup Languaguem) é a linguagem de marcação padrão usada pra ciar e estruturar conteudo da web, ela tem a função de definir o inicio e o fim do documento HTML.

Dentro dessa tag, vocÊ pode colocar elementos como:

  • TITLE - define o titulo da pagna, que aparece na aba do nagegador
  • META - fornece metadados como a descrição da pagna, palavras-chaves para o SEO (otimimização debuusca),, e informações de caracteres.
  • LINKS - usados para vincular arquivos externos, como folhas de estilo CSS.
  • STYLE - permite a inclusão de estilos CSS diretamente no documento.
  • SCRIPT - pode conter ou vincular scripts JavaScript.

a tag HEAD é importante para configurar como a pagna sera interpretada pelos navegadores e como ela aparecera nos motores de busca, mas não é visivel para os usuarios na interface da pagna da web.

HEADY - é uma seção de documentos que cótem informações sobre a págna, mas que não são exibidas diretamente no conteudo visual da pagina.

BODY - em HTML, representa o corpo do documento, ou seja, a parte que contem todo o conteudo visual da pagina web. tudo o que você deseja que o usuario veja e interaja na pagna deve esta dentro da tag BODY, isso inclui textos,tabelas, imagens, links,videos,e qualquer outro que compem a interface da pagna.

IMPUT - em html para criar campos interativos em formularios, onde os usuarios podem inserir dados. o tipo de entrada pode ser alterados com o tributo TYPE,pemintindo criar diferentes tipos de campos, como botoes de envio, campos de textos, caixas de seleção, entre outros.

Exemplos de tag de IMPUT:

PRINCIPAIS TIPOS DE IMPUT

  • TEXT - campo para inserir texto.
  • PASSWORD - campo para inserir senha ( os caracteres ocultos).
  • CHECKBOX - caixa de seleção.
  • RÁDIO - botão de opção (onde o usuario pode escolher apenas uma opçãoentre varias).
  • SUBMIT - botão de envio de formulario.
  • EMAIL - campo para iserir endereço de email
  • DATE - campo para selecionar data
  • NUMBER - campo para numeros

A tag IMPUT não tem tag de fechamento, e ses e seus atributos são usados para definir o tipo de entrada id, nome, valor e outos detalhes sobre o campo.

BUTTON - em html, é usada para criar botões interrativos em uma pagna da web. Diferente da tag IMPUT com o tipoSUBMIT, que cria um botão de envio de formulario, o BUTTON oferece mais flexibilidade, pois pode ser usado tanto para enviar formularios quanto para realizer outras ações, como executar codigo JavaSxrito.

Exemplos basicos de como usar a tag BUTTON:

ATRIBUTOS COMUNS DE BUTTON

  • TYPE - define o tipo de botão, pode ser:
  • SUBMIT - envia o formulario (comportamento padrão).
  • RESET - redefine todos os campos do formulares para valores iniciais.
  • BUTTON - cria um botão comum, que pode ser usado JavaScript.
  • ONCLICK - define a ação ser executada quando o botão for clicado, geralmente associada a unma função JavaScript.
  • DISABLED - desativa o botão, impedindo que ele seja clicado

IMG - em html é usada para exibir imnagens em uma pagna da web. Ela não possui a tag de fechamento, e seu principal atributo é o SRC que especifica o c\minho para imagem a ser exibida.A tag IMG tambem pode incluir outros atributos como ALT, WIDTH, HEIGHT, entre outros, para fornecer informações adicionais sobre a imagem ou controlar exibição.

Exemplos basicos de como usar a tag IMG:

ATRIBUTOS COMUNS DA COMUNS DA TAG IMG:

  • WIDTH - define a largura da imagem.
  • HEIGHT - define a altura da imagem.
  • ALT - fornece uma descrição da imagem para leitores de tela e quando a imagem não pode ser carregada.
  • TITLE - fornece uma dica de ferramenta(texto exibido quando o usuario passa o mouse sobre a imagem).

DIV - em html é usada como um elemento de bloco generico para agrupar outros elementos de estruturar o conteudo de uma pagna. ele não tem feito visual proprio, mas é muito util para organizar e aplicarestilos ( geralmente em css) ou manipular conteudos com JavaSCRIPT.

Exemplos basicos de como usar a tag DIV:

PRINCIPAIS USO DA TAG DIV

  • ESTRUTURAÇÃO DE LAYOUT - dividir a pagna em seçõeaa ouconteineres para aaolicar estilos e organizar conteudos.
  • ESTLIZAÇÃO EM CSS - permitir a aplicações de estilos de manrira eficientea em grupoas de elementos.
  • MANIPULAÇÃO EM JAVASCRIPT - facilitar seleção e manipulação de blocos de conteudo.


BR - em HTML serve para criar uma quebra de linha no conteúdo da página. Ou seja, ela permite que o texto ou conteúdo na página seja dividido em duas linhas, sem a necessidade de criar um novo parágrafo (como seria feito com a tag

). É útil quando você quer iniciar uma nova linha em um local específico sem adicionar espaço extra entre o conteúdo

Exemplos basicos de como usar a tag BR:

OUTROS USOS COMUM

  • dentro de paragrafos, lista, ou outras tags de texto para controlar onde o conteudo se devide em novas linhas
  • ao formatar endereços ou textos em quebra de linhas são necessarias.

A tag BR é uma tag de "auto-fechamento", ou seja, ela não precisa de uma tag de fechamento.

HR - em HTML é usada para criar uma linha horizontal na página, geralmente usada para separar visualmente seções de conteúdo. Essa linha pode ser estilizada com CSS para mudar a cor, espessura, largura, entre outros.

Exemplos basicos de como usar a tag HR:

OUTROS USOS COMUM

  • separa seções ougrupos de conteudo
  • melhor e organizar visual das pagnas.
  • criar uma divisão clara entre diferentes partes de um artigo ou blog.
